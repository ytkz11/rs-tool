# 后台运行

> 远程本地多用户桌面（是一台电脑，不需要两台电脑）

 * 模拟器运行拥有卡顿，性能消耗大等诸多缺点
 * 我们推荐您使用 Windows 自带的远程桌面服务进行该程序
 * 在电脑上直接运行的性能消耗要小于模拟器
